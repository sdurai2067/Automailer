Autoemailer README
Steps to Use:
1. Download Python:
Make sure you have Python installed on your computer. You can download it from python.org.

2. Download the Project:
Download the Autoemailer project files from the repository.

3. Install Dependencies:
Open a terminal or command prompt, navigate to the project folder, and run the following command:

pip install tkinter
This command installs the necessary libraries for the program to work.

4. Run the Program:
Navigate to the project folder containing the autoemailer.py file using the terminal or command prompt.

Run the program by typing:

python sendmails.py (or mail.py)--any one is enough
5. Fill in Information:
You'll see a window pop up.
Provide the file paths for the template file and the CSV file containing the email data.
Choose the appropriate use case from the dropdown menu.
6. Click "Send Emails":
Once you've filled in all the information, click the "Send Emails" button.

7. Done!
The program will start sending emails based on the provided data. You'll receive a notification once all emails are sent.
